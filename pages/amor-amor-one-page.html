Restaurante Amor Amor

Projeto do desenvolvimento de um site institucional para o restaurante Amor Amor, localizado na cidade de Araras, no interior de São Paulo com o objetivo de divulgar sua marca e seus produtos para um número maior de possíveis clientes.

Desenvolvimento

O desenvolvimento foi realizado por meio das seguintes etapas:

HTML 5
CSS 3
Bootstrap
Github
  • Realização de um briefing com o responsável pelo restaurante;
  • Desenvolvimento do wireframe;
  • Codificação do site.
