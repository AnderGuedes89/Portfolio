myLibrary

Projeto do desenvolvimento da prototipagem para um sistema web desktop para a gestão do acervo da biblioteca da Faculdade de Tecnologia de Araras.

Desenvolvimento

O desenvolvimento desse projeto seguiu o paradigma do Design Centrado no Usuário, que por meio de várias ferramentas que busca identificar as necessidades dos usuários e produzir um sistema facilmente utilizável e acessível, visando a satisfação do usuário.

Trabalho em Equipe
  • Brainstorm;
  • Pesquisa quantitativa por meio de um formulário;
  • Desenvolvimento do wireframe;
  • Prototipagem no Figma.

Considerações

A princípio a ideia dessa prototipagem é referente apenas a um sistema web no formato desktop, mas atualmente a grande maioria das pessoas utilizam smartphones para acessar os sites e sistemas, então como uma possível nova feature seria importante a prototipagem desse mesmo sistema na versão mobile.

Para o real desenvolvimento do sistema a abordagem escolhida seria uma API em .net core, que seria consumida por uma aplicação web em Angular e para a versão mobile a API poderia ser consumida por uma aplicação Android, iOS ou Flutter.
